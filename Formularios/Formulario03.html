    Etiqueta o atributo Descripción
    form Creación de formulario
    input Control interactivo y personalizado de formularios
    input type="" Indica el tipo de input a utilizar
    input placeholder="" Se usa como "consejo" de los datos a introducir
    input value="" Se introduce un valor predeterminado al input
    input name="" Se usa para valores internos y accesibilidad
    autofocus Centra el formulario en dicho campo al entrar a la página
    input required Campo obligatorio
    input min="" Valor mínimo a introducir
    input max="" Valor máximo a introducir
    input pattern="" Patrón del input a introducir
    progress Esta etiqueta se utiliza para visualizar el progreso de una tarea.
    meter Con esta etiqueta se pretende representar mediante una barra un rango de valores.
