    Estructura y elementos de RSS 2.0:

    1. Estructura principal de un documento RSS

    <?xml version="1.0" encoding="ISO-8859-1" ?>

    <rss version="2.0">

    <channel>

    <title><!--nombre de nuestro feed-->

    <link><!---la URL de nuestro sitio Web -->

    <description><!-- tipo de contenidos que vamos a incluir en el canal.-->

    <item>

    <title>Título del artículo</title>

    <link>Dirección Web a la que podemos ir para ampliar esta información</link>

    <description>Contenido de esta información</description>

    </item>

    </channel>

    </rss>

    2.Indica cuántos elementos channel puede haber en un documento RSS

    Sólo puede haber un channel en un documento RSS.

    3.Indica los elementos requeridos en channel (son tres)

    Los elementos del channel requeridos son el title, link y description

    4.Indica loselementos secundarios en channel(son 15)

    Los elementos secundarios son language, copyright, managingEditor, webMaster, pubDate, lastBuildDate, category, generator, docs, cloud, ttl, image, textInput, skipHours, skipDays

    5.Elementos item

    Los elementos de item son title, link, description, author, category, comments, enclosure, guid, pubDate, source

    6. Indica el elemento que añade un elemento multimedia. Indica sus tres atributos

    enclosure: url,lenght,type


    Estructura y elementos de Atom 1.0:

    1.Estructura principal de un documento Atom

    <?xml version="1.0" encoding="utf-8"?>

    <feed xmlns="http://www.w3.org/2005/Atom">

    <title>Example Feed</title>

    <link href="http://example.org/"/>

    <updated>2003-12-13T18:30:02Z</updated>

    <author>

    <name>John Doe</name>

    </author>

    <entry>

    <title>Atom-Powered Robots Run Amok</title>

    <link href="http://example.org/2003/12/13/atom03"/>

    <id>urn:uuid:1225c695-cfb8-4ebb-aaaa-80da344efa6a</id>

    <updated>2003-12-13T18:30:02Z</updated>

    <summary>Some text.</summary>

    </entry>

    </feed>

    Plantilla obtenida de Request for Comments: 4287 (rfc4287)
    Plantilla obtenida de

    2.Indica cuántos elementos feed pueden haber en un documento Atom

    Puede haber sólo un elemento feed en el documento atom

    3.Indica los elementos requeridos en el feed

    Los elementos obligatorios son el id, title y updated

    4.Indica los elementos recomendados en feed

    Los elementos recomendados son author, link

    5.Indica los elementos opcionales en feed

    Los atributos opcionales son category, contributor, generator, icon, logo, rights, subtitle

    6.Indica los elementos requeridos en entry

    Los elementos obligatorios son id, title y updated

    7.Indica los elementos recomendados en entry

    Los elementos recomendados son author, link, content y summary

    8.Indica los elementos opcionales en entry

    Los atributos opcionales son category, contributor, rights, source, published
