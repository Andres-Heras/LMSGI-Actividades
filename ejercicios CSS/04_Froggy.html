JUSTIFY-CONTENT (Propiedad de alineación):

Define cómo los elementos flexibles se disponen a lo largo del eje principal en la línea en curso. Puede tomar los siguientes valores:

  • flex-start: Alinea elementos al lado izquierdo del contenedor.
  • flex-end: Alinea elementos al lado derecho del contenedor.
  • center: Alinea elementos en el centro del contenedor.
  • space-between: Muestra elementos con la misma distancia entre ellos.
  • space-around: Muestra elementos con la misma separación alrededor de ellos.
 01

#pond {

display: flex;

justify-content: flex-end

}
02

#pond {

display: flex;

justify-content: center

}
03

#pond {

display: flex;

justify-content: space-around

}
04

#pond {

display: flex;

justify-content: space-between

}

Align-items (Propiedad de alineación)

define cómo los elementos flexibles se disponen a lo largo del eje secundario de la línea en curso.

  • flex-start: Alinea elementos a la parte superior del contenedo
    • flex-end: Alinea elementos a la parte inferior del contenedor.
  • center: Alinea elementos en el centro (verticalmente hablando) del contenedor.
  • baseline: Muestra elementos en la línea base del contenedor
  • stretch: Elementos se estiran para ajustarse al contenedor.
5

#pond {

display: flex;

align-items: flex-end

}
justify-content + align-items 06

#pond {

display: flex;

justify-content:center;

align-items:center;

}
07

#pond {

display: flex;

justify-content:space around;

align-items:flex-end;

}

flex direction (Propiedad de alineación)

establece el eje principal en la que se colocan los elementos flexibles en los contenedores.

  • row Elementos son colocados en la misma dirección del texto.
    • row-reverse: Elementos son colocados en la dirección opuesta al texto.
  • column: Elementos se colocan de arriba hacia abajo en la misma dirección del texto.
  • row-reverse: Elementos son colocados en la dirección opuesta al texto.
  • column-reverse: Elementos se colocan de abajo hacia arriba.
08

#pond {

display: flex;

flex-direction:row-reverse;

}
09

#pond {

display: flex;

flex-direction:column;

}

#pond {

display: flex;

flex-direction:row-reverse;

justify-content:flex-end;

}

#pond {

flex-direction:column;

justify-content:flex-end;

}
12

#pond {

display: flex;

flex-direction:column-revers;

justify-content:space-between;

}
13

#pond {

ORDER

Esta propiedad permite cambiar el orden de los elementos flexibles en el codigo fuente de la página web. Se le pueden dan valores enteros positivos o negativos. El valor predeterminado es 0.

 14

#pond {

display: flex;

}

.yellow{

order:-1;

}
15

#pond {

display: flex;

}

.red{

order:-1;

}

Esta propiedad acepta los mismos valores de align-items (flex-start,flex-end,center,baseline y strech) pero son usados para un elemento específico.

 16

display: flex;

align-items:flex-star;

}

.yellow{

align-self: flex-end

}
17

display: flex;

align-items:flex-star;

}

.yellow{

order:1;

}

flex-wrap

Controla la dirección del eje secundario y la dirección en la que las nuevas líneas se apilan.

  • nowrap Cada elemento se ajusta en una sola línea.
    • wrap: los elementos se envuelven alrededor de líneas adicionales.
  • wrap-reverse: Los elementos se envuelven alrededor de líneas adicionales en reversa.
  • row-reverse: Elementos son colocados en la dirección opuesta al texto.
  • column-reverse: Elementos se colocan de abajo hacia arriba.
18

#pond {

display: flex;

align-items:flex-star;

}

.yellow{

flex-wrap:wrap

}
flex-wrap + flex-direction 19

#pond {

display: flex;

flex-direction:column;

flex-wrap:wrap

}

flex-flow

combina las propiedades flex-direction y flex-wrap para colocar los elementos flexibles.

 20

#pond {

display: flex;

flex-flow:column wrap;

}

align-content (Propiedad de alineación)

para establecer como múltiples líneas están separadas una de otra. Esta propiedad acepta los siguientes valores:

  • flex-start: Las líneas se posicionan en la parte superior del contenedor.
    • wrap: Las líneas se posicionan en la parte inferior del contenedor.
  • center: Las líneas se posicionan en el centro (verticalmente hablando) del contenedor.
  • space-between: Las líneas se muestran con la misma distancia entre ellas.
  • space-around: Las líneas se muestran con la misma separación alrededor de ellas.
  • stretch: Las líneas se estiran para ajustarse al contenedor.

Esto puede ser confuso, pero align-content determina el espacio entre las líneas, mientras que align-items determina como los elementos en su conjunto están alineados dentro del contenedor. Cuando hay solo una línea, align-content no tiene efecto

 21

flex-flow

combina las propiedades flex-direction y flex-wrap para colocar los elementos flexibles.
22

#pond {

display: flex;

flex-wrap:wrap;

align-content:flex-end;

}
align-content + flex-flow + justify-content 23

#pond {

display: flex;

flex-wrap:wrap;

flex-direction:column-reverse;

align-content:center;

}
align-content + flex-flow+justify-content+align-content 24

#pond {

display: flex;

flex-flow:column-reverse wrap-reverse;

justify-content:center;

align-content:space-between;

}

Conceptos aprendidos:

Contenedor flexible: área de un documento diseñado con flexbox (elemento padre) que tendrá en su interior cada uno de los ítems flexibles y adaptables.

Ítem: Cada uno de los hijos flexibles que tendrá el contenedor en su interior.

Eje principal:Eje a lo largo del cual los elementos flexibles se suceden unos a otros. Por defecto, es en horizontal (fila).

Eje secundario: Es el perpendicular a la principal. Si la principal es en horizontal, la secundaria será en vertical, y viceversa.

Flexbox: Módulo de CSS que define un modelo de caja, que tiene capacidad de alinear y justificar elementos (ítems) en los ejes principal y transversal del contenedor flexible, y de distribuir espacio entre elementos flexibles que contiene. Esta capacidad, fue el motivo por que el que se diseñó este modelo unidimensional de layout, porque permite colocar los diferentes elementos de una página, de manera, que se comporten de forma predecible cuando el diseño de la página debe acomodarse a diferentes tamaños de pantalla y diferentes dispositivos.
